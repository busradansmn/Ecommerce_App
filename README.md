# E-Commerce Flutter App
DummyJSON verileriyle çalışan modern e-ticaret uygulaması

## Proje Açıklaması
Bu uygulama; DummyJSON API üzerinden ürünleri dinamik olarak çeken, kullanıcıların ürün arayabildiği, sepet ve favori yönetimi yapabildiği ve kullanıcı girişi bulunan detaylı bir e-ticaret uygulamasıdır.
Şık arayüz, hızlı performans ve temiz mimari ile geliştirilmiştir.

---

## Özellikler
✅ Ürün listeleme (API)  
✅ Ürün detay görüntüleme  
✅ Ürün arama  
✅ Sepete ekleme / çıkarma  
✅ Favorilere ekleme / çıkarma  
✅ Kullanıcı girişi  
✅ Responsive ve modern UI

---

## Kullanılan Teknolojiler

| Alan | Teknoloji |
|------|----------|
| UI | Flutter |
| Dil | Dart |

| Veri Depolama | Hive |
